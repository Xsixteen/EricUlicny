SunTray

Language

Description
Sun Tray displays information about the Sunset in the OSX tray. Information such as time until sunset, time until sunrise, and tomorrows length of day. One big hurdles in the development of this application were implementing the Sunset Algorithm (http://williams.best.vwh.net/sunrise_sunset_algorithm.htm) and learning how to create a Java OSX Application.
Check out the project at https://github.com/Xsixteen/SunTray
